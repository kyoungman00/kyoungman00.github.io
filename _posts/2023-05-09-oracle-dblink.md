# Select Oracle DBlink

Typing bellow command in dbms.   

```sql
select * from dba_db_links;
```


> OWNER : Db link owner.   
> DB_LINK : Db link Object name.   
> USERNAME : User name with permissions.  
> HOST : The database to connected to.
